## 🚀 Key Features

### 1. Tactical Intel AI Assistant (Neural Link)
A fully functional, real-time AI assistant built directly into the UI.
* **Powered by Google Gemini 1.5 Flash API.**
* Features a strict **JSON Intelligence Core** (Vagish Dossier) injected via System Instructions to prevent hallucination and ensure the AI acts as a dedicated personal representative.
* Includes matrix-style typing animations and "neural processing" UI feedback.
* Supports a hidden `/sudo update intel` command for on-the-fly context injection.

### 2. Academic Intel & Timeline
An immersive, glowing vertical timeline detailing educational background:
* B.Tech in CSE (Cybersecurity) @ NMAMIT
* Diploma in CSE @ NMIT
* Designed with tactical glassmorphism and connected data-trace lines.

### 3. Holographic Certificate Deck
A custom-built, 3D rotating carousel that allows users to seamlessly scroll through high-level certifications, including:
* IISc Bangalore Ethical Hacking Workshop
* Mastercard Cybersecurity Job Simulation
* Various Cloud & AI credentials

### 4. Advanced Frontend UI/UX
Designed without heavy UI libraries, relying on custom CSS and DOM manipulation:
* **BMW M-Performance Ignition Base:** An interactive bottom control deck with forged carbon visuals, M-stripes, and "engine rev" hover states.
* **Telemetry HUDs:** Custom SVG layouts for a live Drone Clock and a "Hobbies" technical dashboard mapped with status indicators.
* **Fluid Physics Cursor:** A custom "wavy" cursor trail system that dynamically follows user input, replacing the default browser pointer globally.

---

## 🛠️ Technology Stack
* **HTML5:** Semantic structure and layout backbone.
* **Tailwind CSS:** Rapid, utility-first styling for complex grid layouts, glassmorphism, and responsive design.
* **Vanilla JavaScript (ES6+):** Object-oriented classes managing state, AI asynchronous requests, 3D math for the carousel, and cursor physics.
* **Google Gemini API:** Cloud-based LLM integration with strict system prompting.
